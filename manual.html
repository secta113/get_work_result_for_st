給与明細 自動取得ツール 手順書 (V6.3対応版)

v1.2 (2025-11-13)

このツールは、WEB給与明細サイトに自動ログインし、全期間のサマリーCSVを作成します。ID/PWはPC固有の情報で暗号化されて core/.env に自動保存されます。

1. 必要なもの (Pythonのインストール)

このツールを実行するPCには、**Python** がインストールされている必要があります。

Python (バージョン 3.12 を推奨 3.14~では動作しないことを確認済みです)

インストールされていない場合は、以下の公式サイトからダウンロードしてインストールしてください。

https://www.python.org/downloads/release/python-31210/

重要: インストール時、「Add Python to PATH」 (または「Add python.exe to PATH」) のチェックボックスを必ずオンにしてください。

2. フォルダ構成

ダウンロードしたzipファイルを展開し、ファイルが以下の構成になっていることを確認してください。

プロジェクトフォルダ/
├── apprun.bat    (← 実行ファイル)
├── manual.html   (← この手順書)
├── output/       (← CSV/Log保存先 ※自動作成)
└── core/
    ├── setup.bat     (← ★ 初回に実行)
    ├── app.py      (← アプリ本体)
    ├── main_controller.py
    ├── ... (その他 .py ファイル)
    ├── encryption_utils.py (★ 暗号化ロジック)
    ├── requirements.txt
    └── .env        (← ID/PW(暗号化)自動保存先)

3. 初回セットアップ (PC1台につき1回)

ツールを初めて使う前に、必要なライブラリをインストールします。

  1. 1 core フォルダを開く

    プロジェクトフォルダの中にある core フォルダに移動してください。

  2. 2 setup.bat をダブルクリック

    core フォルダ内にある setup.bat をダブルクリックして実行します。

    （黒いコンソール画面が起動し、自動でライブラリのインストールが始まります。完了まで1〜2分かかる場合があります）

  3. 3 完了を確認する

    「セットアップが正常に完了しました」というメッセージが表示されたら、ウィンドウを閉じてください。

    エラーの場合: 「Python が見つかりません」と表示された場合は、手順1に戻り、Pythonのインストール（特に「Add to PATH」のチェック）を確認してください。

  4. 3 .env ファイルの作成

    core フォルダ内に、.env という名前で新規ファイルを作成し、以下の内容をコピーして貼り付けてください。

    MY_LOGIN_ID=""
MY_PASSWORD=""
LOGIN_COMPANY_CODE="your_company_name"

    （IDとパスワードは空のままで構いません。初回ログイン成功時に自動で暗号化されて保存されます）

4. 実行方法

  1. 1 apprun.bat をダブルクリック

    プロジェクトのルートフォルダ（core フォルダの *外*）にある apprun.bat をダブルクリックします。

    （黒いコンソール画面が起動し、しばらくすると自動でブラウザが開きます）

  2. 2 IDとパスワードを入力

    初回実行時のみ: ブラウザに表示された「ログインID」「パスワード」欄に、給与明細サイトの情報を入力してください。

    2回目以降: 前回正常にログインできたID/PWが自動で入力されています。

  3. 3 「実行」または「全期間スキャン」を押す

    「対象年」を確認し、ボタンを押してください。（通常は「全期間スキャン」を推奨します）

  4. 4 ログイン情報の自動保存

    処理が（ログアウトまで）正常に完了すると、入力したIDとパスワードが core/.env ファイルに自動で暗号化されて保存されます。 これにより、次回の起動が楽になります。

    注意: ログインに失敗した場合は、間違ったID/PWは保存されません。

5. 仕様変更・注意点 (V6.3)

V6.3より、セキュリティ向上のためID/PWの暗号化方式が変更されました。

  • PC固有の暗号化 (マシンロック)
    ID/PWを暗号化する際、PC固有のMACアドレスをキーとして使用します。これにより、core フォルダ（.env を含む）を**別のPCにコピーしても動作しません**。
  • PCを移行（買い替え）する場合
    新しいPCで、再度「初回セットアップ」からやり直す必要があります。（古い .env ファイルは引き継げません）
  • ランダムMACアドレスについて
    ノートPCなどで「ランダムなハードウェアアドレス」機能が有効だと、PCを再起動するたびにMACアドレスが変わり、.env が読めなくなる可能性があります。もしログインが頻繁に失敗する場合は、OSのWi-Fi設定でこの機能をオフにしてください。

6. 結果の確認

処理が成功すると、output フォルダ内に以下のファイルが作成（または更新）されます。

  • 年間サマリー_全期間.csv
  • app_log.log (動作ログ)
  • debug_requests_network_log.txt (詳細な通信ログ)

年間サマリー_全期間.csv ファイルはExcelなどで開くことができます。

7. 終了方法

アプリを終了するには、apprun.bat を実行した際に開いた**黒いコンソール画面**を選択し、Ctrl キーを押しながら C キーを押してください。

（または、コンソールの「閉じる」ボタン（×）を押しても終了できます）
